
# Attoya PHP Framework Scaffold


## Scaffold

This is an example use of the Attoya PHP/JS Frameworks


## Installation

- Attoya PHP Framework needs to be installed at `lib/vendor/attoya`
- Attoya JS Framework (static directory) needs to be installed at `lib/vendot/static/attoya`
- Smarty3 PHP Template Engine needs to be installed at `lib/vendor/smarty3`
